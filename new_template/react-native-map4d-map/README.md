# react-native-map4d-map

Map4dMap SDK for React Native

## Installation

```sh
npm install react-native-map4d-map
```

## Usage

```js
import { Map4dMapView } from "react-native-map4d-map";

// ...

<Map4dMapView color="tomato" />
```
